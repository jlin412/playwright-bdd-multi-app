# Accessibility Checklist

**Cross-cutting coverage gate for UI and mobile accessibility.** The UI checklist
references this file rather than repeating it.

## Keyboard
- Full keyboard navigation
- Logical tab order
- Visible focus indicator
- Escape closes modal
- Enter/Space activates controls
- No keyboard traps

## Screen Reader
- Meaningful labels
- Proper headings
- ARIA labels where needed
- Error messages announced
- Dynamic content announced
- Modal focus management

## Forms
- Labels associated with inputs
- Required fields announced
- Validation errors clear
- Error summary where appropriate
- Instructions available

## Visual
- Color contrast
- Text resizing
- Zoom support
- No color-only meaning
- Responsive readability
- Icons have labels or alt text

## Components
- Buttons
- Links
- Dropdowns
- Checkboxes
- Radio buttons
- Date pickers
- Tables
- Modals
- Toasts
- Tabs
- Accordions

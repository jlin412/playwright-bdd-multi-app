# Performance Checklist

**Cross-cutting coverage gate for UI, API, mobile, and IoT performance.** The domain
checklists reference this file rather than repeating it.

## UI Performance
- Page load time
- Slow network
- Large data sets
- Search response
- Pagination speed
- File upload/download
- Form submission
- Dashboard rendering
- Repeated clicks
- Browser memory usage

## API Performance
- Response time
- Throughput
- Latency
- Payload size
- Pagination performance
- Concurrent requests
- Rate limiting
- Timeout behavior
- Retry behavior
- Downstream dependency delay

## Mobile Performance
- App startup time
- Screen transition time
- Battery usage
- Memory usage
- CPU usage
- Network switching
- Offline recovery

## IoT Performance
- Command latency
- Telemetry delay
- Device reconnect time
- Cloud sync delay
- Firmware update duration
- Offline recovery time

# Performance Testing Knowledge

Use this checklist for UI, API, mobile, and IoT performance coverage.

## UI Performance
- Page load time
- Slow network
- Large data sets
- Search response
- Pagination speed
- File upload/download
- Form submission
- Dashboard rendering
- Repeated clicks
- Browser memory usage

## API Performance
- Response time
- Throughput
- Latency
- Payload size
- Pagination performance
- Concurrent requests
- Rate limiting
- Timeout behavior
- Retry behavior
- Downstream dependency delay

## Mobile Performance
- App startup time
- Screen transition time
- Battery usage
- Memory usage
- CPU usage
- Network switching
- Offline recovery

## IoT Performance
- Command latency
- Telemetry delay
- Device reconnect time
- Cloud sync delay
- Firmware update duration
- Offline recovery time

## Output Guidance
Planning should identify performance-sensitive flows.
Manual test cases may include basic response/load expectations.
Automation planning should identify candidates for automated performance smoke checks, not full load testing unless requested.
